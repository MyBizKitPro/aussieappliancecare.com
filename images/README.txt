SAVE YOUR PHOTOS IN THIS FOLDER  (REJ Services/images/)
=========================================================

The website looks for these exact file names. To use your own photo,
just OVERWRITE the matching file with the same name and the site updates
automatically — no code changes needed.

TIP: keep the SAME file name and extension (.jpg). If you must use a
different extension (.png/.webp), the matching <img src="..."> in
index.html has to be updated too — easiest to just export as .jpg.

TIPS
----
- Export photos as JPG and keep each under ~300KB so the site stays fast
  on mobile.
- Keep a consistent look across all photos (similar lighting / colour
  grade, ideally with teal or warm tones) — that's what makes it feel
  premium and cohesive.
